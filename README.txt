Breaking News displays important posts on every page.

== Description ==

Breaking News plugin will display important posts marked as Breaking News on every page.

== Installation ==

1. Upload `breaking-news.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
